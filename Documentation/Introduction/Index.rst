What does it do?
================

This extension aims to provide more or less useful ViewHelpers, Utilities and other classes
either as an example or for direct use in your own extension

When installed, a fluid namespace

..  attention::

    Do not forget to set extension's version number in :file:`Settings.cfg` file,
    in the :code:`release` property.
    It will be automatically picked up on the cover page by the :code:`|release|` substitution.
